$ git config --global user.name - устанавливает имя
$ git config --global user.email -устанавливает имейл
$ git init - создает репозиторий локальный
Я еше склонировал репо с гита с помощью $ git clone
$ git pull обьеденяет удаленный и локальный
$ git add добовляет все файлы если в конце (.) либо можно указать определенный файл
$ git commit -m добовляет комит (понимаю что это но не могу описать) при изменениях файла ставятся
комиты понятные тебе и другому разработчику
$ git push пушаться файлы на удаленный репа
$ git branch (имя ветки) создать новую ветку
$ git switch -c (имя ветки) переключает на выбранную ветку
$ git merge (имя ветки) вносит изменения из указанной ветки в ту в которой находимся
За ответами на вопросы иду сюда Git documentation
