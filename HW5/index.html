Типы селекторов

* - Выбирает все элементы. По элементу - пример: селектор input выберет все элементы input. По классу - .red выберет все элементы с соответствующим классом По идентификатору - селектор #green выберет элемент с идентификатором green Можно обратиться к нескольким элементам перечислить их через , - div, span выберет оба элемента селектор ul > li - выберет только дочерние элементы li h2 + p - выберет первый элемент p, который находится непосредственно после элемента h2 Знак : позволяет выбрать элементы, основываясь на информации, которой нет в дереве элементов. Знак :: позволяет выбрать вещи, которых нет в HTML.
